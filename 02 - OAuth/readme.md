# Séance 2 - Mise en pratique avec le protocole OAuth2

_Tout en s'entrainant sur deno, nous allons découvrir le protocole OAuth 2. On
va créer un site qui va lister les serveurs discord d'une personne identifiée_

Prérequis :

- Bases sur Deno (javascript ou typescript)
- Savoir comprendre les documents fournis

Objectif de l'atelier :

- Savoir utiliser deno (ex : dépendance permettant de créer un serveur web)
- Découvrir Oauth2 et l'API http de Discord.
